# Hospital Management System

A production-ready Flask-based Hospital Management System with role-based access for doctors and patients.

## Features

- 🔐 **Authentication System** - Secure login/register with password hashing
- 👨‍⚕️ **Doctor Dashboard** - Manage appointments, approve/reject requests
- 👤 **Patient Dashboard** - Book appointments, view history
- 📅 **Appointment Scheduling** - Real-time slot availability checking
- 🌙 **Dark Theme UI** - Modern, professional interface
- 📱 **Responsive Design** - Works on all devices

## Tech Stack

- **Backend**: Python Flask
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript
- **Authentication**: Werkzeug Security
- **Deployment**: Gunicorn ready
